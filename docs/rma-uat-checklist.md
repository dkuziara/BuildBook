# RMA UAT Checklist

## Register and navigation

- Open the RMA Register and confirm active records load.
- Filter by customer, product, serial number, status, and linked Build Record state.
- Open Board view and Reports from the register.

## Create and edit

- Create a new RMA with required intake fields only.
- Create an RMA using Planner migration traceability fields.
- Open the detail page and edit intake, workflow, and fault sections.

## Checklist and status flow

- Mark a checklist item complete and then open again.
- Add a custom checklist item.
- Change status from Booked In to Work In Progress.
- Change status to On Hold with a reason and note.

## Linking and protection

- Link an RMA to a Build Record.
- Confirm linked Build Record secrets are not exposed through the RMA module.

## Reporting and trends

- Open reports by status and by customer or product.
- Confirm repeat return and duration information appears where expected.
